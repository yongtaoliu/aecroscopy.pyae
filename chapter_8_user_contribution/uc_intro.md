User contributions are a vital part of the AEcroscopy community. This chapter collects workflows, scripts, and experiment designs shared by researchers who have built on top of AEcroscopy in their own labs and projects.

There are many ways to contribute, including:

- A **control interface for a different microscopy platform**, joining Cypher and Nanosurf in Chapter 2.
- A **control interface for a different instrument** that works alongside the microscope (e.g. a temperature controller, an external DAQ, a robotic sample changer).
- A new **mode** — a composed, ready-made experiment pipeline, like `Auto3DPFM` (Chapter 5).
- A new **automated workflow** — a hand-built experiment sequence using existing interfaces, like the pulse workflows in Chapter 6.
- A new **autonomous workflow** — an AI/ML-driven exploration strategy, like those in Chapter 7.
- Anything else you've built on AEcroscopy that others in the community would find useful.

We welcome contributions from the community. If you're interested in contributing any of the above, please reach out to Yongtao Liu at liuy3@ornl.gov, or youngtaoliu@gmail.com, or via [LinkedIn](https://www.linkedin.com/in/yong-tao-liu/).
